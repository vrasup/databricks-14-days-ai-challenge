# Databricks 14 Days AI Challenge 

This repository documents my daily learning and hands-on work as part of the  
**Databricks 14 Days AI Challenge**.

## About the Challenge

The challenge is designed to help beginners build practical skills in Databricks, Apache Spark, and modern data engineering.

The 14-day challenge focuses on:
- Building strong fundamentals in Databricks and Lakehouse architecture
- Working with structured and semi-structured data
- Practicing SQL and PySpark in real-world scenarios
- Understanding how Databricks supports AI and analytics workflows
